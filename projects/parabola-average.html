Fluid simulation made using manim

Average value of a Function via Fluid Dynamics

This page hosts your Manim animation and the accompanying Python code. The scene plots \(f(x) = (x - 3)^2\), fills the region under the curve with particles, and then runs a particle based fluid simulation. An incompressible fluid will naturally settle at the average value of \(f(x)\).

The Lagrangian Fluid Simulator was built from Seb Lague's Fluid-Sim repository and there is an accompanying video going over the mechanics/updating process here: Seb's accompanying YouTube Video. I had to incorporate a few tweaks,

  • The initializtion process had to change to fill the area under a curve.
  • I also added random (normally distributed) horizontal "jitters", little force bumps. I did this because, without this feature, it was possible to get "particle stacking" along walls, and relatively large gaps/irregularities in the final state. These little horizontal jitters act like tapping a glass of sand, it helps it settle and fill in cracks.

To use this Manim animation: If you have not made manim animations before, the actual "scene" is made in the method at the bottom of the code titled, FluidSPHScene(Scene). In that scene you'll want to scroll down to SPHSimulation2D and adjust parameters such as, dt, pressure_multiplier, near_pressure_multiplier, viscosity_strength, and collision_damping to get a feel for how they affect the mechanics first. I use a large near_pressure_multiplier because I don't want any particles overlapping.

Once you get a feel for how the parameters impact the scene, I would encourage you to begin playing with defining your own function and interval. This is also within the FluidSPHScene, you'll see
def f(x):
# return 1 + 0.5 * x
# return 1 + 3 * np.sin(np.pi*x/W)
return (x-3)**2
Where you can also see my other test functions of a linear and a sinusodal.
