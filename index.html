What this starter includes

  • Static HTML, CSS, and JavaScript
  • Python syntax highlighting
  • Responsive code and media layout
  • Local MP4 support for short clips
  • Optional embedded YouTube videos

How to use it

A simple pattern for each simulation

1. Add your Python file

Place the source code in examples/ and point a simulation page at it with a data-code-src attribute.

2. Add a short MP4

Put small demonstration clips in assets/videos/ and use the HTML <video> element with controls.

3. Link a longer explanation

Embed a YouTube lecture or talk when you want extra context beyond the short simulation clip.
